# SnapShare 📸

**SnapShare** is a futuristic, pixel-inspired image sharing platform with a high-fidelity glassmorphism UI.

## 🚀 Repostiory
Built with the MERN stack: [https://github.com/evil-sandeep/snapshare.git](https://github.com/evil-sandeep/snapshare.git)

## ✨ Features
- **Pixel-Art Aesthetic**: Silkscreen typography with neon accents.
- **Glassmorphism**: Premium frosted-glass cards and navigation.
- **Framer Motion**: Smooth page transitions and element animations.
- **Custom Ripple Effect**: High-fidelity button interactions.
- **Micro-Interactions**: Hover, click, and loading states for a premium SaaS feel.

## 🛠️ Stack
- **Frontend**: React (Vite-powered) + Framer Motion
- **Backend**: Node.js + Express
- **Database**: MongoDB (via Mongoose)
- **Styling**: Vanilla CSS with Design System Variables

## 📦 Getting Started

1. **Clone the project**:
   ```bash
   git clone https://github.com/evil-sandeep/snapshare.git
   cd snapshare
   ```

2. **Install dependencies**:
   ```bash
   npm run install-all
   ```

3. **Configure Environment**:
   Update `backend/.env` with your MongoDB and Cloudinary credentials.

4. **Launch Application**:
   ```bash
   npm start
   ```
